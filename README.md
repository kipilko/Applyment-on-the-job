# Yuppy

Corporate website for Yuppy - a global outsourcing and software development company.

## About

Yuppy delivers cutting-edge digital solutions for businesses of all sizes. We specialize in end-to-end product development from strategy to launch and support.

## Quick Start

1. Open `index.html` in your browser
2. Navigate through sections: About, Services, Apply, and Contact

## Sections

- **About** - Company mission, values, and what makes us unique
- **Services** - Tech requirements and benefits offered
- **Apply** - Job application form for Full-Stack Junior Developer position
- **Contact** - Get in touch with our team

## Tech Stack

- HTML5, CSS3, JavaScript
- React or similar JS frameworks
- Node.js / Express.js
- MySQL, PostgreSQL, or MongoDB
- Python (optional)

## Job Opening

## Contact
